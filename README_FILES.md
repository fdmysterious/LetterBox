Index des fichiers :
=======================
## Extensions

* .h   : Fichiers headers : Contient les déclarations de fonction ainsi que les constantes de configuration.
* .cpp : Fichiers sources : Contient le code des fonctions à proprement parler.
	
## Liste des fichiers

* pins.h		  : Contient les associations entre les entrées/sorties et les numéros de pin de l'Arduino ; Contient également la fonction d'initialision des modes.
* sleep.{h,cpp}  : Gère la fonction de mise en veille de l'appareil
* detect.{h,cpp} : Gère la fonction de détection du courrier et de batterie faible.
